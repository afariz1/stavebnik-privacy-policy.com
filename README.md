Stavebnik Privacy Policy

# Privacy Policy
This application Staveník (hereinafter referred to as the "Application") is developed for the purposes of Jan Opěla.
# 1. Data Collection and Use
The Application collects specific data only when you use the attendance features:
Location Data: When you click "Start Attendance" (or Stop Attendance), the Application captures your current geographic coordinates. Also when you borrow or return tools outside of storage, the Application captures your current geographic coordinates.
Purpose: This data is used solely for verifying the place of work and place where tools were borrowed for internal attendance records.
Storage: This information is securely stored in our database hosted by Supabase.
# 2. Access to Device Features
To provide its core functionality, the Application requires access to:
Location Services: To determine your position at the moment of recording attendance. This data is only captured upon your explicit action (clicking the button), not in the background.
Camera - to scan qr codes to assign to the tools for borrowing.
Notifications - to send notification when user forgets to end attendance in the evening.
Optionally storage - to set up pictures for tools to better recognize them.
# 3. Third-Party Services
We use Supabase as our backend service provider to store and process the data mentioned above. Supabase handles data in accordance with industry security standards. We do not sell or share this data with any other third parties or advertising networks.
# 4. Data Retention
Attendance and location data are kept only for as long as necessary for internal administrative purposes or as required by local labor laws.
# 5. Your Rights
Users have the right to request access to their stored data or request its deletion by contacting the administrator.
